Vercel 404 fix checklist

1) Project Root Directory must point to this app folder.
   - If your repo root contains portfolio_upload_ready/, set Root Directory = portfolio_upload_ready
   - If you uploaded the app files directly at repo root, set Root Directory = .

2) Framework Preset: Vite
3) Build Command: npm run build
4) Output Directory: dist
5) After changing settings, Redeploy from Vercel.

If the site still shows 404 at the root URL, the most likely cause is wrong Root Directory or a failed build in Vercel logs.
